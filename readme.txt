An add-on for Contact Form 7 that provides an updated and compact data viewing option.

== Description ==

Contact Form 7 - Compact DB is an advanced add-on for Contact Form 7 which provide a very compact way of viewing all the submitted data beside the "Advanced Settings" tab.

It's a very light weight plugin that way it won't effect anything on page speed.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/cf7cdb` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Goto Contact->Contact Forms-> click 'Edit' on the contact form you want to view.
1. Beside 'Additional Settings' a new tab called 'Submitted data' is there.


== Frequently Asked Questions ==

= Is this compatable for all Contact form 7 vesions?  =

Yes. It will work with all versions of cf7.

= Previous submitted data are not showing =

All submitted data before adding this plugin will not show.
